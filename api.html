API Documentation

Integrate live Zynther Tiers rankings into your own client applications or discord bots.

GET /api/v1/overall

Fetches the complete overall leaderboard including player points, titles, regions, and sub-category tiers.

{
  "rank": ,
  "name": "",
  "title": "",
  "points": ,
  "region": "",
  "tiers": {
    "sword": "",
    "smp": "",
    "pot": ""
  }
}
GET /api/v1/category/{category}

Fetches the tier list for a specific combat category. Replace {category} with: vanilla, uhc, pot, nethop, smp, sword, axe, or mace.

{
  "tier": ,
  "players": [
    { "name": "", "region": "" }
  ]
}
GET /api/v1/player/{username}

Looks up a specific player by their Minecraft username. Returns all their ranking data across every category.

{
  "name": "",
  "title": "",
  "points": ,
  "region": "",
  "tiers": { ... }
}
